1 - após  instalado o MySQL em sua máquina, agora instale o módulo 
mysql.connector utilizando o sistema de gerenciador de pacotes pip.
Para isso, abra o terminal do seu sistema operacional e 
digite pip install mysql-connector-python ou para evitar 
possíveis erros, insira no final a versão do mysql instalado em sua máquina 
usando esse comando: pip install mysql-connector-python==8.0.13 

2 - carreque o arquivo bd.sql
